# linkedin-agent

A practical LinkedIn product-posts ranking agent with a working browser demo and an n8n workflow starter.

## Files
- `app/linkedin-agent-demo.html`
- `n8n/linkedin-product-posts-agent-n8n.json`

## Browser demo
Open the HTML file locally, load the sample dataset or paste your own CSV, and review the top-ranked product and leadership posts.

## n8n workflow
Import the JSON into n8n and replace the mock collector with Google Sheets, Airtable, webhook input, or another approved source.
